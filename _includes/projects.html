Legend of Zelda (NES) Rebuilt

  • Unity

This Legend of Zelda Game is a remake of a dungeon in the original Legend of Zelda Game. This project served as an introduction to the Unity Game Engine where I learned and progressed my understanding of Unity. In addition to recreating the game, I learned a lot about good game design. With this knowledge, I implemented a new level into the game with a completely new mechanic not found in the original game.
